

## 简介

本站是基于layui Fly社区建立。代码完全开源，包含后端

## 友情链接

本站体验地址：https://blog.frontblog.top

Layui： http://fly.layui.com/

后端地址：https://gitee.com/lsfcxz/koa_api

## 技术栈

前端：Vue、Vuex、VueRouter、LayUI

后端：NodeJs、Koa

数据库：MongoDB、Redis

## 使用方式

- 安装依赖

```
npm install
```

- 运行

```
npm run serve
```

- 构建

```
npm run build
```

- ESLine

```
npm run lint
```

> TIP：需要在本地运行时需要搭配后端使用，修改config/index里面的baseurl即可
