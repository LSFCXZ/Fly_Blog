

## 简介

一个社区博客应用，主要使用vue2.x、layui和elementui，目前实现了以下功能：

- 登录注册、找回密码、修改密码、修改基本信息
- 发帖、回帖、评论、点赞
- 编辑帖子、签到积分体系
- 首页分类
- 更多功能请点击体验地址：https://blog.frontblog.top

如果该应用对你有所帮助，欢迎start！！！！

## 使用方式

- 安装依赖

```
npm install
```

- 运行

```
npm run serve
```

- 构建

```
npm run build
```

- ESLine

```
npm run lint
```

PS：需要在本地运行时需要搭配后端使用，修改config/index里面的baseurl即可
