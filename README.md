## 使用方式

- 安装依赖

```
npm install
```

- 运行

```
npm run serve
```

- 构建

```
npm run build
```

- ESLine

```
npm run lint
```

> TIP：需要在本地运行时需要搭配后端使用，修改config/index里面的baseurl即可
